# Project Blueprint: Astro Portfolio

## Overview
A modern, content-focused portfolio website built with Astro.js, focusing on performance and minimalist design.

## Features & Styles
- **Tech Stack:** Astro.js, Node.js 20.
- **Design:** Modern, minimalist, featuring IBM Plex Mono font for a terminal feel.
- **Interactivity:** Islands architecture for minimal JS.
